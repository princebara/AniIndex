# AniIndex

AniIndex is a responsive anime search tool built with HTML, CSS, and JavaScript, powered by the Jikan REST API. It allows users to search for anime, browse results, view information, and view of recently viewed titles.

You can view the site: https://princebara.github.io/AniIndex/

<img width="1918" height="970" alt="Screenshot 2026-07-06 172621" src="https://github.com/user-attachments/assets/61e1213e-7589-48dc-981d-20942e7b7770" />

Features:
- Search for anime
- View information for each anime
- Display score, status, episodes, duration, release year, source, genres, and synopsis
- Recently viewed sidebar
- Recent searches persist using Local Storage
- Light/Dark mode toggle
- Responsive design for desktop, tablet, and mobile devices

Built With:
- HTML5
- CSS3
- JavaScript (ES6+)
- Jikan REST API

This project was built as a learning exercise to gain practical experience with:

Fetching data from REST APIs
Working with asynchronous JavaScript (async / await)
Managing application state
Using Local Storage for persistent data
Responsive layouts
Organizing a vanilla JavaScript project

And i definitely learned a lot, and will continue to make more projects in the future documenting my progress.

AniIndex uses the Jikan REST API, an unofficial REST API for MyAnimeList.
